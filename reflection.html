What I changed in Assignment 2

In Assignment 1, my page had the text but it did not really look like a magazine page yet. For Assignment 2, I changed the layout so the article is split into columns using CSS instead of tables. This made it look closer to the printed version and also made the code cleaner.

I also changed the colours and spacing to make the page feel more like photocopied print. I used an off-white background instead of a bright white screen, and I kept most of the colours black and dark red so the page would feel more like a punk zine and less like a modern website.

For JavaScript, I added a small jQuery effect to make parts of the page appear gradually. I did not want the effect to be too dramatic because the main focus should still be on the article itself.
