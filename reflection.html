Assignment Reflection

REFLECTION

Looking back at Assignment 1

In Assignment 1, the easiest part for me was setting up the basic HTML files and linking the pages together. Once I understood the structure of an HTML document, it was not too difficult to add headings, paragraphs, and links.

The hardest part was recreating the Punk Planet page itself. I found it difficult to decide how to organise the article in HTML and how to make it look similar to a printed magazine page. At that stage, I focused more on getting the text into the page correctly than on making the design look accurate.

What I changed in Assignment 2

For Assignment 2, I improved the structure of the website and spent more time on the visual layout. I kept the three-page format from Assignment 1, but I changed the styling so the pages would feel more like a printed fanzine instead of a plain HTML document.

One important improvement was changing the article layout into columns using CSS. This helped the page look more like a magazine and also followed the assignment instruction to avoid using tables for layout. I also made the homepage and reflection page more consistent with the same overall style.

My style choices

I chose an off-white background instead of a bright white one because it feels closer to paper than to a modern website. I mainly used black text and lines, because I wanted the design to feel simple, bold, and slightly rough in a punk zine style.

I also changed the typography so that the body text feels more like a printed article. The headings are larger and more dramatic, while the article text is more compact so it can sit in columns like a magazine spread. I tried not to make the layout too polished, because the original print style feels more raw and independent.

Final thoughts

Overall, I think Assignment 2 is a clear improvement on Assignment 1. The first version had the main content, but this version is closer to the look of an actual fanzine page because it uses layout, typography, spacing, and colour more carefully.

If I had more time, I would continue refining the details of the design and experiment with making the page even closer to the original printed version. However, I think this version shows a better understanding of how HTML, CSS, and JavaScript can work together in a small website.
